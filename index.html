CSS Positioning Cheat Sheet

Static

An element with position: static; occupies its normal position in the document flow. Block-level elements will take up an entire line, while inline elements will take up only their content space. No offset or height/width parameters can be set.

CSS code Example -Static 1

.static__1 {
position: static;
background-color: orange;
}

Relative

An element with position: relative; is positioned relative to its 'normal' position in the document flow. The top, right, bottom, and left rules can be used to offset these elements from their normal positon.

Relative positoned elements are still in the normal document flow.

Relative 1 - This div is in its normal document flow position
Relative 2 -This div is offset from the top by 30px, and the left by 40px.

CSS Code Example - Relative 2

.static__2 {
position: relative;
top: 30px;
left: 40px;
background-color green;
}

Absolute

An element with position: absolute; will be positioned relative to its nearest positioned ancestor (instead of relative to the normal document flow). If an absolute positioned element has no positoned ancestors, it will use the document flow.

Absolute positioned elements are removed from the normal document flow, and can overlap other elements

Absolute Child

CSS Code Example - Absolute Parent & Absolute Child

.absolute__parent {
position: relative;
width: 500px;
height: 200px;
background-color orange;
}

.absolute__child {
position: absolute;
width: 250px;
height: 100px;
top: 100px;
left: 250px;
background-color green;
}

Fixed

An element with position: fixed; has a fixed postion relative to the viewport. This means it always stays in the same place regardless of scrolling.

A fixed position element is removed from the normal flow, so can overlap.

Notice the element in the lower-right portion of the screen - it stays there no matter where you scroll.

This element is fixed in place with position: fixed;

Code Example - Fixed Element on Bottom Right

.fixed__example {
position: fixed;
bottom: 10px;
right: 10px;
width: 200px;
height: 50px;
background-color: lightgreen;
}

Sticky

An element with position: sticky; is positioned with a mix of relative and fixed, depending on scroll position

The element is positioned position: relative until a given offset position is met in the viewport - then it 'sticks' by flipping to position: fixed
